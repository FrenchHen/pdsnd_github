## US Bikeshare Data

### Description
This is the repo for **Udacity's US Bikeshare Data Project**.
The script runs with Python 3 and is designed to explore the Bike sharing data of three US cities: Washington, New York City and Chicago.

### Files used
- README.md
- bikeshare.py
- chicago.csv
- new_york_city.csv
- washington.csv
